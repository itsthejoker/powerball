The purpose of this template is to create single-page simple websites that do one thing and one thing only.

Use the template at `index.html` to build the app that the user has requested.

## Functionality

The goal is to create a small, static one-page website that does one thing. Functionality is
intentionally extremely limited; do not add extra features.

## UI

Keep the UI as simple as possible. Bootstrap CSS and JS are already available; use them. The content
should be centered in the screen with an emphasis on readability and usability. Ideally, on mobile,
the content should not even need to be scrolled. Do not add flashy custom CSS unless asked for by the
user.

## Coding style

Keep code clean and simple. Do not add emojis, do not add comments unless necessary, and do not add
extra error handling (like try/catch blocks that just pass). The end product will be a project that
is so simple that a JS error should basically be impossible.
